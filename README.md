# App gerador de senhas randômicas

### Aplicativo gerador de senhas aleatorias com a opção de escolher dentre 5 a 15 caracteres.

# Requisitos
  - Node 12+
  - Expo 
  - Android Studio ( Para emular um aparelho )

# Getting started
  ``` npm install ``` ou ``` yarn install ```

# Iniciando Projeto
  ```expo start ```
